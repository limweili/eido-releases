# Eido releases

This is Eido's **public-but-unadvertised** binary release repository. Public
means that links may be forwarded or discovered; it is not access control.

No Eido download is available from this repository yet. Friends Alpha will
open only when one complete release contains both required targets: macOS arm64
and Windows x64. A release missing either platform is incomplete and must not be
promoted.

When downloads appear, every release will include platform signatures,
checksums, release notes, and machine-readable integrity information. Eido does
not collect automatic telemetry. Support reports must exclude saved Ideas,
Card text, prompts, assets, API keys, credentials, and private filesystem paths.

Eido's source repository is private. This repository publishes release policy
and, later, signed binary artifacts; it does not publish or license the private
source code.
